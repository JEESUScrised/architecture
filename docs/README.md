# Nova Studio — сайт на БЭМ

Учебный многостраничный сайт (3 страницы, на главной — 3 экрана прокрутки).

## БЭМ в проекте

| Блок | Элементы / модификаторы |
|------|-------------------------|
| `header` | `header__inner`, `header__logo`, `header__logo-accent` |
| `nav` | `nav__list`, `nav__link`, `nav__link--active`, `nav__list--open` |
| `screen` | `screen__inner`, `screen__title`, `screen--hero`, `screen--services` |
| `card` | `card__icon`, `card__title`, `card__text` |
| `button` | `button--primary`, `button--outline` |
| `footer` | `footer__inner`, `footer__link` |
| `form` | `form__group`, `form__input`, `form__textarea` |

## Страницы

- `index.html` — главная (hero, услуги, CTA)
- `about.html` — о студии
- `contacts.html` — форма и контакты

## Локальный запуск

Откройте `index.html` в браузере или через Live Server.
