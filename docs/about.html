О студии

Nova Studio — учебный многостраничный сайт для демонстрации методологии БЭМ (Блок, Элемент, Модификатор). Имена классов отражают структуру: например, header__inner — элемент блока header.

Принципы БЭМ в проекте

Блок — независимый компонент (header, card, footer).
Элемент — часть блока (nav__link, card__title).
Модификатор — вариант блока или элемента (screen--hero, button--primary).

Команда

Блок team с элементами team__member, team__name, team__role.
