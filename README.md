# SOLID — примеры на Python 3

| Файл | Принцип |
|------|---------|
| `srp.py` | Single Responsibility |
| `ocp.py` | Open/Closed |
| `lsp.py` | Liskov Substitution |
| `isp.py` | Interface Segregation |
| `dip.py` | Dependency Inversion |

В коде есть комментарии по каждому принципу.

```bash
python srp.py
python ocp.py
python lsp.py
python isp.py
python dip.py
```
